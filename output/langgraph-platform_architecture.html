01
Agent Loop
Andrew Ng's 4 patterns, expressed as a graph
Perceive · Reason · Act · Evaluate · Persist
① Perceive
· state loaded from checkpointer
· thread-scoped, automatic
② Reason
· LLM call → AIMessage(tool_calls)
· Planning = writes a plan first
③ Act
· ToolNode executes tool_calls
· Multi-Agent = delegates to sub-graph
④ Evaluate
· conditional_edge(state) → next
· Reflection = routes back to Reason
⑤ Persist
· state snapshot → checkpointer
· crash-safe, resumable
Loops until done:  perceive → reason → act → evaluate → persist → ↻ loop
